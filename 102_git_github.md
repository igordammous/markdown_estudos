# Git e Github
Git é case sensitive, portanto letra maiúscula e minúscula se diferenciam.
## Comandos
* `ls` – listar o que está na página
* `cd` – entrar em pasta (“cd Desktop/” – entra na pasta desktop dentro do Git)
* `pwd` – mostra o caminho absoluto da página onde você está
* `ls -a` – mostra arquivos ocultos
* `cd ..` – vai para a pasta acima da pasta onde estava anteriormente
* `mkdir` – cria um novo diretório
* `cat` – mostra o conteúdo do arquivo (“cat comandos.bash.txt” – mostra o que tem dentro do arquivo comandos.bash.txt)
* `touch` – cria um arquivo (“touch comandos.bash.txt” – cria um arquivo txt chamado comandos.bash)
* `git init` . – permite a pasta se tornar um repositório para git e assim é possível versionar os arquivos dessa pasta
* `git status` – status do repositório
* `git add` – adiciono o que eu quero que faça parte da versão do commit
* `git commit` -m "Primeiros comandos bash" – commit com a mensagem que está entre aspas
* `git log` – mostra todos os commits feitos
* `rm – rf` – comando para excluir a pasta(“rm -rf .git/” – apaga a pasta oculta git[.git])
